내일 평균 · MAPE
기본 오차 = 전월 기준 MAPE(낮을수록 정확). 전체 누적 오차는 툴팁에 함께 표시합니다. backtest

모델 × 시간대 히트맵

행=모델, 열=시각. 색이 진할수록 고가. 세로로 색이 갈리면 모델 간 이견이 큼.

전월 오차 (MAPE)

모델별 전월 평균 오차. 짧을수록(시안) 정확, 길수록(앰버) 부정확.

내일 시간대별 가격표

모델별 24시간 예측 가격(p50). 앙상블과 챔피언 모델은 색으로 강조했습니다.

실측 SMP 대비 모델 예측 (백테스트 2026-05)

굵은 흰 선 = 실제 SMP(기준선). 각 색선 = 모델 예측(p50). 두 선이 가까울수록 정확합니다. 아래 슬라이더로 기간을 좁혀 볼 수 있습니다. x축은 일자·시간대(hour-ending).

일자별 예측 점수

각 날짜의 실측 대비 예측 정확도. 점수 = 100 - MAPE, 높을수록 좋습니다.
